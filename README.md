### About

Full-stack developer building at the intersection of Web3, AI, and security. Creator of [**web3-agent-kit**](https://github.com/ulsreall/web3-agent-kit) — an open-source Python SDK for autonomous blockchain AI agents, published on PyPI with 986 tests and 7+ chain support.

Currently building [**Mancing DAO**](https://t.me/mancingdao) — a Web3 community focused on airdrop farming, DeFi security, and onchain strategy. Open to collaboration and full-time opportunities.

<sub>**Python · TypeScript · Rust · Go · Solidity** &nbsp;|&nbsp; **EVM · Solana · Base · Celo · Farcaster**</sub>
